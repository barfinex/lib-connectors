# Barfinex Connectors

Standardized exchange integration layer with REST/WS connectors.

## ✨ Features
- Written in TypeScript
- Reusable across Barfinex ecosystem
- Designed for modular microservices

## 📦 Installation
```bash
npm install @barfinex/connectors
```

## 🚀 Usage
```ts
import { Example } from "@barfinex/connectors";
```
